# 划译 MVP 设计

> 这是 v0.1.x 历史基线；欧路生词本增量见
> [v0.2.0 设计](./2026-07-12-eudic-wordbook-design.md)，YouTube 字幕取词增量见
> [YouTube 字幕取词设计](./2026-07-18-youtube-caption-selection-design.md)。

## 目标

在普通英文网页中双击或拖选文本，显示豆包风格的轻量浮层。单词、短语和单句可选择中文解释
或英译中；多句段落仅提供翻译。第一版通过本机 Codex CLI 复用 ChatGPT 订阅，不接入普通
OpenAI API。

## 内容规范

词汇翻译展示语境中文义、词性、适用时的英美音标、常见搭配、原段落例句和 3–5 个相似
词。词汇解释展示语境含义、原形/构词、核心义、3–5 个同义词和语境搭配。相似词与同义词
只展示英文、词性和中文义。句子解释展示主干、关键表达、翻译和语境作用。

## 交互规范

鼠标完成选区时，工具条、加载状态和结果卡贴近选区且水平中心对齐鼠标释放位置；键盘选区
回退到选区矩形。结果卡宽 420px、最大高 70vh，可拖动并限制在视口内。卡片只保留标题、
拖动把手和关闭按钮，不提供朗读、复制、反馈、刷新或后续对话。新选区、关闭或 Escape 会
取消旧请求。

浮层状态统一为 `idle -> actions -> loading -> result/error -> closed`；错误重试回到
`loading`，迟到结果不能重新打开已关闭浮层。结果卡使用开源无依赖的原生 DOM 与 Shadow
DOM，宽度不超过 420px、高度不超过视口 70%，内容超出时只在卡片内部滚动。拖动位置也属于
状态机数据，并同时支持指针拖动与方向键微调。

## 隐私规范

只发送选区及所在段落的有限上下文，不发送页面元数据，不保存历史。每次 Codex 调用均使用
ephemeral 会话、只读沙箱和结构化输出。
