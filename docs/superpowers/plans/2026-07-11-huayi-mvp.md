# 划译 MVP Implementation Plan

> **For agentic workers:** execute tasks in order with TDD, focused verification, and one
> Conventional Commit per task.

**Goal:** Build the governed Huayi monorepo, Chrome extension, Native Messaging host, ephemeral
Codex provider, macOS installer, and verification suite.

**Architecture:** Manifest V3 content script and service worker communicate with a macOS Node host
through Native Messaging. Extension and host share only `@huayi/protocol`; the host invokes
`codex exec --ephemeral` behind `AnalysisProvider`.

**Tech Stack:** pnpm, TypeScript, Vite, Zod, Vitest, Playwright, Codex CLI.
